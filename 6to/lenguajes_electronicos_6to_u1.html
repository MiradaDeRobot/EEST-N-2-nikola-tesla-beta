1. Registros (Estructuras)

Un registro, conocido en C como `struct`, es un tipo de dato compuesto que permite agrupar múltiples variables de diferentes tipos bajo un solo nombre. Es ideal para representar entidades del mundo real que tienen varias características.

Definición y Uso

Se definen con la palabra clave `struct` y se accede a sus miembros (las variables internas) con el operador punto (`.`).

Ejemplo: 
#include <stdio.h>
#include <string.h>

// Se define la estructura para un Alumno
struct Alumno {
    char nombre[50];
    int legajo;
    float promedio;
};

int main() {
    // Se declara una variable del tipo 'struct Alumno'
    struct Alumno alumno1;

    // Se asignan valores a sus miembros
    strcpy(alumno1.nombre, "Juan Perez");
    alumno1.legajo = 12345;
    alumno1.promedio = 8.5;

    // Se accede a los miembros para imprimirlos
    printf("Nombre: %s\n", alumno1.nombre);
    printf("Legajo: %d\n", alumno1.legajo);
    printf("Promedio: %.2f\n", alumno1.promedio);

    return 0;
}

2. Arrays (Arreglos)

Un array es una colección de elementos del mismo tipo almacenados en ubicaciones de memoria contiguas. Permiten manejar grandes cantidades de datos de forma organizada.

Arrays Unidimensionales (Vectores)

Son listas de elementos. Se accede a cada elemento mediante un índice, que en C siempre comienza en 0.

Arrays Bidimensionales (Matrices)

Son tablas de elementos organizados en filas y columnas. Se necesitan dos índices para acceder a un elemento: el de la fila y el de la columna.

Ejemplo: 
#include <stdio.h>

int main() {
    // Declaración de una matriz de 2 filas y 3 columnas
    int matriz[2][3] = { {1, 2, 3}, {4, 5, 6} };
    int i, j;

    // Se recorre la matriz con dos bucles 'for' anidados
    for (i = 0; i < 2; i++) { // Bucle para las filas
        for (j = 0; j < 3; j++) { // Bucle para las columnas
            printf("%d ", matriz[i][j]);
        }
        printf("\n"); // Salto de línea al final de cada fila
    }

    return 0;
}

3. Punteros

Un puntero es una variable especial que no almacena un dato, sino una dirección de memoria. Son una de las características más potentes (y complejas) de C, y son fundamentales para el manejo eficiente de la memoria y el acceso a hardware.

Operadores Clave

  • Operador de Dirección (`&`): Devuelve la dirección de memoria de una variable.
  • Operador de Indirección (`*`): Se usa para declarar un puntero y para acceder al valor que está almacenado en la dirección de memoria a la que apunta.

4. Funciones: Pasaje por Valor y por Referencia

Las funciones son bloques de código reutilizables que realizan una tarea específica. En C, los argumentos se pueden pasar a una función de dos maneras.

Pasaje por Valor

Es el método por defecto en C. La función recibe una copia del valor de la variable. Cualquier modificación que la función haga a ese valor es local y no afecta a la variable original fuera de la función.

Pasaje por Referencia

Se logra pasando un puntero (la dirección de memoria) a la función. La función no recibe una copia, sino un acceso directo a la variable original. Por lo tanto, cualquier modificación que realice a través del puntero sí afectará a la variable original.

5. Archivos

El manejo de archivos permite a un programa leer y escribir información de forma persistente en el disco duro.

Archivos de Texto vs. Archivos Binarios

  • Archivos de Texto (`.txt`): Almacenan la información como caracteres legibles por humanos. Son fáciles de inspeccionar, pero menos eficientes.
  • Archivos Binarios (`.dat`, `.bin`): Almacenan los datos en el mismo formato binario que la computadora usa en la memoria RAM. Son más rápidos, eficientes y ocupan menos espacio, pero no son legibles directamente.
Funciones Clave (`stdio.h`):
- `fopen()`: Abre un archivo.
- `fclose()`: Cierra un archivo.
- `fprintf()` / `fscanf()`: Para escribir/leer en archivos de texto con formato.
- `fwrite()` / `fread()`: Para escribir/leer bloques de datos en archivos binarios.
