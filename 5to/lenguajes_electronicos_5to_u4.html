1. Estructuras Condicionales

Las estructuras condicionales permiten que un programa tome decisiones y ejecute diferentes bloques de código según si se cumple o no una determinada condición.

La Sentencia `if`, `else if` y `else`

Es la estructura de decisión más fundamental.

  • `if`: Ejecuta un bloque de código solo si la condición es verdadera.
  • `else`: Ejecuta un bloque de código si la condición del `if` es falsa.
  • `else if`: Permite encadenar múltiples condiciones.
Ejemplo: 
// Se declara una variable entera llamada 'nota' y se le asigna el valor 7.
int nota = 7;

// Si la nota es mayor o igual a 7, se ejecuta este bloque.
if (nota >= 7) {
    printf("Aprobado\n");
} 
// Si la primera condición no se cumple, se evalúa esta. Si la nota es mayor o igual a 4.
else if (nota >= 4) {
    printf("Recupera\n");
} 
// Si ninguna de las condiciones anteriores es verdadera, se ejecuta este bloque.
else {
    printf("Desaprobado\n");
}

La Sentencia `switch`

Es una alternativa al `if-else if-else` que resulta más clara y eficiente cuando se necesita comparar una única variable con múltiples valores posibles.

  • `case`: Define uno de los posibles valores que puede tomar la variable.
  • `break`: Es crucial. Detiene la ejecución dentro del `switch` para que no se sigan evaluando los demás casos.
  • `default`: Es opcional. Se ejecuta si ninguno de los `case` coincide con el valor de la variable.
Ejemplo: 
// Se declara una variable de tipo caracter llamada 'opcion'.
char opcion = 'B';

// Se evalúa el valor de la variable 'opcion'.
switch (opcion) {
    // Si 'opcion' es igual a 'A', se ejecuta este código.
    case 'A':
        printf("Se seleccionó la opción A.\n");
        break; // Se sale del switch.

    // Si 'opcion' es igual a 'B', se ejecuta este código.
    case 'B':
        printf("Se seleccionó la opción B.\n");
        break; // Se sale del switch.

    // Si 'opcion' es igual a 'C', se ejecuta este código.
    case 'C':
        printf("Se seleccionó la opción C.\n");
        break; // Se sale del switch.

    // Si 'opcion' no coincide con ningún caso anterior.
    default:
        printf("Opción no válida.\n");
        break;
}

2. Estructuras de Repetición (Bucles)

Los bucles o lazos de repetición permiten ejecutar un bloque de código múltiples veces, ya sea un número determinado de veces o mientras se cumpla una condición.

Bucle `while`

Ejecuta un bloque de código mientras una condición sea verdadera. La condición se evalúa antes de cada iteración, por lo que el bloque podría no ejecutarse nunca.

Ejemplo: Contar hasta 5. 
// Se inicializa un contador en 1.
int contador = 1;

// Mientras el contador sea menor o igual a 5, el bucle se repetirá.
while (contador <= 5) {
    // Imprime el valor actual del contador.
    printf("%d\n", contador);

    // Incrementa el contador. Es crucial para evitar un bucle infinito.
    contador++; 
}

Bucle `do...while`

Es similar al `while`, pero la condición se evalúa después de cada iteración. Esto garantiza que el bloque de código se ejecute al menos una vez.

Ejemplo: Pedir una contraseña hasta que sea correcta. 
// Se definen las variables para la clave correcta y la ingresada.
int clave_correcta = 1234;
int clave_ingresada;

// El bloque 'do' se ejecuta al menos una vez.
do {
    // Pide al usuario que ingrese la clave.
    printf("Ingrese la clave: ");
    scanf("%d", &clave_ingresada);

// El bucle se repetirá mientras la clave ingresada sea distinta a la correcta.
} while (clave_ingresada != clave_correcta);

// Cuando se sale del bucle, significa que la clave es correcta.
printf("Acceso concedido.\n");

Bucle `for`

Es ideal para situaciones donde se conoce de antemano el número de repeticiones. Su estructura compacta incluye la inicialización de un contador, la condición de permanencia y el incremento del contador.

Ejemplo: Imprimir los números del 0 al 4. 
#include <stdio.h>

int main() {
    // Se declara una variable 'i' que servirá de contador.
    int i;

    // 1. Inicialización: i = 0 (se ejecuta una sola vez al inicio).
    // 2. Condición: i < 5 (se evalúa antes de cada repetición).
    // 3. Incremento: i++ (se ejecuta después de cada repetición).
    for (i = 0; i < 5; i++) {
        // Imprime el valor de 'i' en cada iteración.
        printf("%d\n", i);
    }

    return 0;
}

3. Manipulación de Cadenas de Caracteres (`string.h`)

En C, una "cadena de caracteres" o string no es un tipo de dato nativo, sino que se representa como un array de caracteres (`char`) que termina con un carácter especial nulo (`\0`) para indicar su fin.

La Biblioteca `string.h`

Para trabajar con estas cadenas, C proporciona la biblioteca `string.h`, que contiene funciones esenciales para su manipulación.

  • `strlen(cadena)`: Devuelve la longitud de la cadena (sin contar el carácter `\0`).
  • `strcpy(destino, origen)`: Copia la cadena `origen` en la cadena `destino`.
  • `strcat(destino, anexo)`: Concatena (une) la cadena `anexo` al final de la cadena `destino`.
  • `strcmp(cadena1, cadena2)`: Compara dos cadenas alfabéticamente. Devuelve 0 si son iguales, un valor negativo si `cadena1` es menor que `cadena2`, y un valor positivo si es mayor.
Ejemplo: 
#include <stdio.h>
#include <string.h>

int main() {
    // Se declara un array de char con espacio suficiente.
    char saludo[30] = "Hola";

    // Se declara otra cadena.
    char nombre[] = " Juan";
    
    // Se concatena la cadena 'nombre' al final de 'saludo'.
    strcat(saludo, nombre);
    
    // Se imprime la cadena resultante.
    printf("Cadena final: %s\n", saludo);

    // Se imprime la nueva longitud de la cadena.
    printf("Longitud: %d\n", strlen(saludo));
    
    return 0;
}
