1. Estructura de un Programa en C

Todo programa en C, sin importar su complejidad, sigue una estructura básica. Comprender sus partes es el primer paso para aprender a programar.

El Programa Mínimo: "Hola, Mundo"

Este es el programa más simple que se puede escribir en C, y nos muestra las partes esenciales:

// 1. Directivas del pre-procesador
#include <stdio.h>

// 2. Función principal (main)
int main() {
    // 3. Cuerpo de la función
    printf("Hola, Mundo\n"); // Instrucción
    
    // 4. Valor de retorno
    return 0;
}
  1. Directivas del Pre-procesador: Instrucciones que se ejecutan antes de la compilación. `#include ` le dice al compilador que incluya la biblioteca estándar de entrada/salida.
  2. Función `main()`: Es el punto de entrada obligatorio de todo programa en C. La ejecución siempre comienza aquí.
  3. Cuerpo de la Función: Encerrado entre llaves `{}`, contiene las instrucciones que se ejecutarán. Cada instrucción termina con un punto y coma `;`.
  4. Valor de Retorno: La instrucción `return 0;` indica al sistema operativo que el programa terminó correctamente.

2. Tipos de Datos, Variables y Constantes

Para poder trabajar con información, necesitamos almacenarla en la memoria de la computadora. Las variables y constantes son los contenedores que usamos para ello, y los tipos de datos le dicen al compilador qué clase de información vamos a guardar.

Tipos de Datos Fundamentales

  • `int`: Para almacenar números enteros (sin decimales).
  • `float`: Para números de punto flotante (con decimales) de precisión simple.
  • `double`: Para números de punto flotante de doble precisión (más decimales y rango más amplio que float).
  • `char`: Para almacenar un único carácter (letra, número o símbolo).

Variables y Constantes

  • Variables: Son espacios de memoria con un nombre, cuyo valor puede cambiar durante la ejecución del programa. Se deben declarar antes de usarse: `tipo_de_dato nombre_variable;`. Ejemplo: `int edad = 25;`
  • Constantes: Son valores que no pueden ser modificados una vez definidos. Se declaran con la palabra clave `const`. Ejemplo: `const float PI = 3.1416;`
  • Ámbito (Scope): Las variables declaradas dentro de una función (como `main`) son locales a esa función. Las declaradas fuera de cualquier función son globales y accesibles desde todo el programa.

3. Operadores

Los operadores son símbolos que le indican al compilador que realice operaciones matemáticas o lógicas específicas.

Tipos de Operadores

  • Aritméticos: `+` (suma), `-` (resta), `*` (multiplicación), `/` (división), `%` (módulo o resto de la división).
  • Relacionales: Se usan para comparar valores. Devuelven 1 (verdadero) o 0 (falso). `==` (igual a), `!=` (distinto de), `<` (menor que), `>` (mayor que), `<=` (menor o igual que), `>=` (mayor o igual que).
  • Lógicos: Se usan para combinar expresiones lógicas. `&&` (Y lógico), `||` (O lógico), `!` (NO lógico).

4. Pre-procesador y Bibliotecas

El pre-procesador es una herramienta que se ejecuta antes que el compilador. Su trabajo es procesar directivas especiales en el código fuente que comienzan con el símbolo `#`.

Directivas y Bibliotecas

  • `#include`: Es la directiva más común. Le dice al pre-procesador que copie y pegue el contenido de otro archivo (llamado archivo de cabecera o header, con extensión `.h`) en el código fuente.
  • Bibliotecas (Libraries): Son colecciones de funciones pre-compiladas que podemos usar en nuestros programas. Para usar una función de una biblioteca, primero debemos incluir su archivo de cabecera correspondiente.
    • Biblioteca Estándar: Viene con el compilador de C. Ejemplos: `` para entrada/salida (`printf`, `scanf`), `` para funciones matemáticas (`sqrt`, `pow`).
    • Biblioteca de Usuario: Son archivos `.h` que nosotros mismos creamos para organizar nuestro código en módulos reutilizables.
  • `#define`: Se usa para definir constantes simbólicas y macros. El pre-procesador reemplaza todas las ocurrencias de la constante por su valor antes de la compilación. Ejemplo: `#define PI 3.14159`
