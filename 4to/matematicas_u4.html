1. Modelización de Fenómenos del Mundo Real

La modelización matemática es el proceso de traducir un problema del mundo real a un lenguaje matemático. Consiste en identificar las variables importantes, establecer relaciones entre ellas y crear ecuaciones o funciones que describan el fenómeno. Esto nos permite analizar, predecir y resolver el problema de forma sistemática.

Pasos para Modelizar un Problema

  1. Comprender el Problema: Leer atentamente el enunciado e identificar la pregunta que se debe responder.
  2. Identificar las Variables: Determinar qué cantidades son conocidas (datos) y cuáles son desconocidas (incógnitas). Asignarles símbolos (x, y, etc.).
  3. Plantear las Ecuaciones: Traducir las relaciones descritas en el problema a ecuaciones o inecuaciones matemáticas.
  4. Resolver el Modelo: Utilizar métodos algebraicos o gráficos para encontrar la solución del sistema planteado.
  5. Interpretar y Verificar: Traducir la solución matemática de vuelta al contexto del problema original y comprobar si tiene sentido.

2. Ecuaciones e Inecuaciones

Ecuaciones

Una ecuación es una igualdad matemática entre dos expresiones que contiene una o más incógnitas. Resolver una ecuación consiste en encontrar el valor (o los valores) de las incógnitas que hacen que la igualdad sea verdadera.

Inecuaciones

Una inecuación es una desigualdad entre dos expresiones. En lugar de un signo de igual (=), utiliza símbolos como < (menor que), > (mayor que), ≤ (menor o igual que) o ≥ (mayor o igual que). La solución de una inecuación no es un único valor, sino un conjunto de valores o un intervalo.

Ejemplo: Resolver \(2x + 3 > 9\)
\(2x > 9 - 3\)
\(2x > 6\)
\(x > 3\)
La solución son todos los números reales mayores que 3. En notación de intervalo: \((3, \infty)\).

3. Sistemas de Ecuaciones Lineales

Un sistema de ecuaciones es un conjunto de dos o más ecuaciones con varias incógnitas que deben satisfacerse simultáneamente. Un sistema de 2x2 tiene dos ecuaciones y dos incógnitas.

Resolución Analítica

Los métodos analíticos buscan encontrar el valor exacto de las incógnitas mediante manipulación algebraica.

Método de Sustitución

Se despeja una incógnita en una de las ecuaciones y se reemplaza su valor en la otra ecuación.

Ejemplo: Resolver el sistema:
1) \(2x + y = 7\)
2) \(x - y = 2\)
  1. Despejamos 'y' de la ecuación 2: \(y = x - 2\).
  2. Sustituimos esta expresión en la ecuación 1: \(2x + (x - 2) = 7\).
  3. Resolvemos para 'x': \(3x - 2 = 7 \implies 3x = 9 \implies x = 3\).
  4. Reemplazamos el valor de 'x' en la ecuación despejada: \(y = 3 - 2 \implies y = 1\).
  5. Solución: \((x=3, y=1)\).

Método de Igualación

Se despeja la misma incógnita en ambas ecuaciones y se igualan las expresiones resultantes.

Ejemplo: Resolver el sistema:
1) \(2x + y = 7\)
2) \(x - y = 2\)
  1. Despejamos 'y' de la ecuación 1: \(y = 7 - 2x\).
  2. Despejamos 'y' de la ecuación 2: \(y = x - 2\).
  3. Igualamos ambas expresiones: \(7 - 2x = x - 2\).
  4. Resolvemos para 'x': \(9 = 3x \implies x = 3\).
  5. Reemplazamos 'x' en cualquiera de las ecuaciones despejadas: \(y = 3 - 2 \implies y = 1\).
  6. Solución: \((x=3, y=1)\).

Método de Suma y Resta (Reducción)

Se multiplican las ecuaciones por números convenientes para que los coeficientes de una de las incógnitas sean opuestos, y luego se suman las ecuaciones para eliminarla.

Ejemplo: Resolver el sistema:
1) \(2x + y = 7\)
2) \(x - y = 2\)
  1. Los coeficientes de 'y' ya son opuestos (+1 y -1). Sumamos las dos ecuaciones directamente.
  2. \( (2x + y) + (x - y) = 7 + 2 \).
  3. Simplificamos: \(3x = 9 \implies x = 3\).
  4. Reemplazamos el valor de 'x' en cualquiera de las ecuaciones originales, por ejemplo en la 2: \(3 - y = 2 \implies y = 1\).
  5. Solución: \((x=3, y=1)\).

4. Resolución Gráfica y Discusión de Soluciones

Cada ecuación lineal de dos variables representa una recta en el plano cartesiano. La solución de un sistema de 2x2 corresponde al punto (o puntos) de intersección de estas rectas.

Tipos de Soluciones

Sistema Compatible Determinado

Las rectas se cortan en un único punto. Hay una única solución.

Sistema Compatible Indeterminado

Las dos ecuaciones representan la misma recta. Hay infinitas soluciones.

Sistema Incompatible

Las rectas son paralelas y nunca se cortan. No hay solución.

Discusión de Soluciones: Al intentar resolver un sistema analíticamente, si llegamos a una contradicción (ej. \(0=5\)), el sistema es incompatible. Si llegamos a una identidad (ej. \(0=0\)), el sistema es compatible indeterminado.
