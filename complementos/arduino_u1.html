Unidad 1: Introducción a la Plataforma Arduino y Entorno de Desarrollo (IDE)

Programación en Arduino

1. ¿Qué es Arduino?

Arduino es una plataforma de hardware y software de código abierto, diseñada para facilitar la creación de proyectos de electrónica interactiva. Se compone de una placa de desarrollo (hardware) que se programa utilizando un Entorno de Desarrollo Integrado (IDE) (software).

El Hardware: La Placa Arduino Uno

La placa más popular para empezar es el Arduino Uno. Sus partes principales son:

  • Microcontrolador (ATmega328P): Es el cerebro de la placa, donde se ejecuta nuestro programa.
  • Pines Digitales (0-13): Pueden ser configurados como entradas o salidas de dos estados (ALTO/BAJO).
  • Pines Analógicos (A0-A5): Permiten leer voltajes variables de sensores.
  • Conector USB: Sirve para alimentar la placa y para cargarle los programas desde la PC.
  • Botón de Reset: Reinicia la ejecución del programa.

Otros Pines y Conectores Importantes

  • Pines de Alimentación:
    • VIN: Para alimentar la placa con una fuente externa (ej. una batería de 9V).
    • 5V y 3.3V: Pines que proveen una salida de tensión regulada para alimentar sensores y otros componentes.
    • GND: Pines de tierra o masa (0V), la referencia común para todos los voltajes.
  • Pines de Comunicación:
    • Serial (RX/TX): Los pines 0 (RX - recepción) y 1 (TX - transmisión) se usan para la comunicación serie con la computadora u otros dispositivos.
    • I2C (SDA/SCL): Los pines A4 (SDA) y A5 (SCL) se usan para el protocolo de comunicación I2C, muy útil para conectar múltiples sensores con solo dos cables.

2. Otros Modelos de Placas Arduino

Aunque el Arduino Uno es el más popular, existen muchos otros modelos, cada uno adaptado a diferentes necesidades de tamaño, potencia o número de pines.

Arduino Nano

Es una versión compacta del Uno, ideal para montar en protoboards y para proyectos donde el espacio es limitado. Tiene casi las mismas capacidades que el Uno.

Arduino Mega 2560

Una placa mucho más grande y potente. Ofrece 54 pines digitales, 16 entradas analógicas y más memoria, ideal para proyectos complejos como impresoras 3D o robots avanzados.

Arduino Leonardo

Su microcontrolador tiene comunicación USB nativa, lo que le permite a la placa actuar como un dispositivo de interfaz humana (HID), como un teclado o un ratón.

Arduino Pro Mini

Una versión minimalista, sin conector USB ni pines soldados. Es extremadamente pequeña y de bajo consumo, diseñada para ser integrada de forma permanente en un proyecto.

Arduino LilyPad

Una placa con forma circular y pads grandes diseñados para ser cosidos con hilo conductor. Es la placa ideal para proyectos de e-textiles y tecnología vestible (wearables).

3. El Entorno de Desarrollo (IDE)

El IDE de Arduino es un programa simple que se instala en la computadora y nos permite escribir, compilar y subir nuestro código a la placa Arduino. El lenguaje de programación es una versión simplificada de C++.

Estructura de un Programa (Sketch)

Todo programa de Arduino, llamado "sketch", tiene dos funciones principales que son obligatorias:

void setup() {
  // Este código se ejecuta una sola vez, al principio.
  // Aquí se configuran los pines y se inician las comunicaciones.
}

void loop() {
  // Este código se ejecuta en un bucle infinito,
  // una y otra vez, mientras la placa esté encendida.
  // Aquí va la lógica principal de nuestro programa.
}

4. "Hola Mundo" en Arduino: El Sketch "Blink"

El primer programa que se suele hacer en cualquier plataforma de hardware es hacer parpadear un LED. En Arduino, este sketch se llama "Blink" y ya viene incluido en los ejemplos del IDE.

Análisis del Código

Este programa hace parpadear el LED integrado en la placa Arduino, que está conectado internamente al pin 13.

// La función setup se ejecuta una vez cuando se presiona reset o se enciende la placa
void setup() {
  // Inicializa el pin digital 13 como una salida.
  pinMode(13, OUTPUT);
}

// La función loop se ejecuta una y otra vez para siempre
void loop() {
  digitalWrite(13, HIGH);   // Enciende el LED (HIGH es el nivel de voltaje alto)
  delay(1000);              // Espera un segundo (1000 milisegundos)
  digitalWrite(13, LOW);    // Apaga el LED haciéndolo BAJO
  delay(1000);              // Espera un segundo
}
Pasos para subir el código:
1. Conecta tu Arduino a la PC.
2. En el IDE, ve a `Herramientas > Placa` y selecciona "Arduino Uno".
3. Ve a `Herramientas > Puerto` y selecciona el puerto COM al que está conectado.
4. Haz clic en el botón con la flecha hacia la derecha ("Subir").
