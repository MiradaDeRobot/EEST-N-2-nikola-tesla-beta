Unidad 2: Programación de los pines GPIO con Python

Programación con Raspberry Pi

1. ¿Qué son los Pines GPIO?

La Raspberry Pi no es solo una computadora pequeña; también es una potente herramienta para la electrónica. Su principal característica para esto es el cabezal de 40 pines GPIO (General Purpose Input/Output). A diferencia de los puertos USB, estos pines nos dan un control directo y de bajo nivel sobre el hardware, permitiéndonos conectar LEDs, sensores, motores y otros componentes electrónicos.

Programación con Python y la Librería RPi.GPIO

El lenguaje más popular para controlar los pines GPIO es Python, gracias a su simplicidad y a la potente librería RPi.GPIO, que ya viene preinstalada en Raspberry Pi OS. Esta librería nos proporciona funciones sencillas para configurar los pines y leer o escribir en ellos.

2. Actividades Prácticas con Salidas Digitales

Actividad 1: Blink Básico

Este script hace parpadear un LED conectado al pin GPIO 17.

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
led_pin = 17
GPIO.setup(led_pin, GPIO.OUT)

try:
    while True:
        GPIO.output(led_pin, GPIO.HIGH)
        time.sleep(1)
        GPIO.output(led_pin, GPIO.LOW)
        time.sleep(1)
except KeyboardInterrupt:
    GPIO.cleanup()

Actividad 2: Semáforo Simple

Este código simula un semáforo usando tres LEDs (rojo, amarillo y verde) conectados a los pines GPIO 17, 27 y 22.

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
led_rojo = 17
led_amarillo = 27
led_verde = 22

GPIO.setup([led_rojo, led_amarillo, led_verde], GPIO.OUT)

try:
    while True:
        # Verde
        GPIO.output(led_verde, GPIO.HIGH)
        time.sleep(5)
        GPIO.output(led_verde, GPIO.LOW)
        # Amarillo
        GPIO.output(led_amarillo, GPIO.HIGH)
        time.sleep(2)
        GPIO.output(led_amarillo, GPIO.LOW)
        # Rojo
        GPIO.output(led_rojo, GPIO.HIGH)
        time.sleep(5)
        GPIO.output(led_rojo, GPIO.LOW)
except KeyboardInterrupt:
    GPIO.cleanup()

Actividad 3: Efecto "Auto Fantástico"

Este código enciende y apaga una secuencia de 6 LEDs (conectados del pin 4 al 9) para crear un efecto de barrido.

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
led_pins = [4, 17, 27, 22, 5, 6]

for pin in led_pins:
    GPIO.setup(pin, GPIO.OUT)

def knight_rider():
    for pin in led_pins:
        GPIO.output(pin, GPIO.HIGH)
        time.sleep(0.05)
        GPIO.output(pin, GPIO.LOW)
    for pin in reversed(led_pins):
        GPIO.output(pin, GPIO.HIGH)
        time.sleep(0.05)
        GPIO.output(pin, GPIO.LOW)

try:
    while True:
        knight_rider()
except KeyboardInterrupt:
    GPIO.cleanup()

Actividad 4: Generando Sonido con un Zumbador

Podemos generar un tono simple encendiendo y apagando un pin muy rápidamente. Esto crea una onda cuadrada que el zumbador convierte en sonido.

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
buzzer_pin = 23
GPIO.setup(buzzer_pin, GPIO.OUT)

def beep(duration):
    period = 1.0 / 440 # Frecuencia de 440Hz (Nota La)
    delay = period / 2
    cycles = int(duration * 440)
    for i in range(cycles):
        GPIO.output(buzzer_pin, GPIO.HIGH)
        time.sleep(delay)
        GPIO.output(buzzer_pin, GPIO.LOW)
        time.sleep(delay)

try:
    print("Presiona Ctrl+C para salir")
    while True:
        beep(0.5) # Emite un tono de medio segundo
        time.sleep(1) # Espera un segundo

except KeyboardInterrupt:
    GPIO.cleanup()

3. Entradas Digitales: Leyendo un Pulsador

Ahora vamos a leer información del mundo exterior. Usaremos un pulsador para que nuestro programa reaccione a una acción del usuario.

Código en Python para Leer un Botón

Este script enciende un LED mientras un pulsador (conectado al pin GPIO 18) está presionado.

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)

led_pin = 17
button_pin = 18

GPIO.setup(led_pin, GPIO.OUT)
GPIO.setup(button_pin, GPIO.IN, pull_up_down=GPIO.PUD_DOWN)

try:
    while True:
        if GPIO.input(button_pin) == GPIO.HIGH:
            GPIO.output(led_pin, GPIO.HIGH)
        else:
            GPIO.output(led_pin, GPIO.LOW)
        time.sleep(0.1)

except KeyboardInterrupt:
    GPIO.cleanup()
Resistencias Pull-up / Pull-down: Son necesarias en las entradas digitales para asegurar que el pin tenga un estado definido (ALTO o BAJO) cuando el pulsador no está presionado, evitando lecturas "flotantes" o erráticas. La Raspberry Pi nos permite activar estas resistencias internamente por software.
