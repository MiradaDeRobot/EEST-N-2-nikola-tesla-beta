Unidad 3: Variables, Animaciones y Clones

Introducción a la Programación con Scratch

1. Variables: La Memoria de Nuestros Programas

Hasta ahora, nuestros programas pueden reaccionar a eventos, pero no pueden "recordar" información. Para eso, usamos variables. Una variable es como una caja con una etiqueta donde podemos guardar un dato (un número, texto, etc.) para usarlo más tarde.

Creando y Usando Variables

En la categoría Variables (naranja oscuro), podemos crear nuestras propias variables. Al crear una, aparecerán nuevos bloques para manipularla:

  • Fijar [variable] a [valor]: Asigna un valor inicial a la variable (ej. fijar "puntos" a 0).
  • Cambiar [variable] por [valor]: Suma (o resta) un valor al que ya tiene la variable (ej. cambiar "puntos" por 1).
Variables "Para todos los objetos" vs. "Solo para este objeto": Al crear una variable, Scratch te da a elegir. Una variable "global" (para todos) puede ser leída y modificada por cualquier Sprite. Una variable "local" (solo para este objeto) pertenece a un único Sprite.
Actividad: Añadir Puntuación y Vidas al Juego del Gato y el Ratón
  1. Crea una variable global llamada "Puntos".
  2. En el script del Ratón, al inicio, "fijar Puntos a 0".
  3. Dentro del condicional "Si ¿tocando a Sprite1?", añade el bloque "cambiar Puntos por 1".
  4. Ahora, añade un nuevo Sprite que sea el "malo" (ej. un murciélago).
  5. Crea una variable local para el Gato llamada "Vidas".
  6. En el script del Gato, al inicio, "fijar Vidas a 3".
  7. Añade un condicional al Gato: "Si ¿tocando a Murciélago?, entonces cambiar Vidas por -1 y esperar 1 segundo".

2. Animaciones con Disfraces

La animación en Scratch se basa en el concepto de disfraces. Un Sprite puede tener múltiples imágenes o "disfraces". Al cambiar rápidamente entre ellos, creamos la ilusión de movimiento, como en los dibujos animados tradicionales.

El Editor de Disfraces

Haciendo clic en la pestaña "Disfraces" (junto a "Código"), puedes ver, editar y crear nuevos disfraces para tu Sprite. Muchos Sprites de la biblioteca ya vienen con varios disfraces listos para animar.

Actividad: Hacer que un Personaje Vuele
  1. Elige un Sprite que tenga disfraces de movimiento, como un murciélago ("Bat") o un pájaro.
  2. Crea un script que comience con la bandera verde.
  3. Dentro de un bucle "por siempre", añade el bloque "siguiente disfraz" de Apariencia.
  4. Añade un pequeño bloque "esperar 0.1 segundos" de Control dentro del bucle. Sin esta pausa, el cambio de disfraz sería demasiado rápido para verse.
  5. Añade bloques de movimiento para que el personaje se desplace por el escenario mientras mueve sus alas.

3. Interacción Avanzada: Mensajes

A veces necesitamos que la acción de un Sprite active un script en otro Sprite. Para esto, Scratch utiliza un sistema de mensajes. Un Sprite puede "enviar" un mensaje, y otros Sprites pueden tener scripts que se activan "al recibir" ese mensaje.

Bloques de Mensajes

Se encuentran en la categoría Eventos.

  • Enviar [mensaje]: Emite un mensaje a todos los Sprites del proyecto.
  • Al recibir [mensaje]: Es un bloque de inicio que ejecuta el script cuando recibe un mensaje específico.
Actividad: Nivel Superado
  1. En nuestro juego del Gato y el Ratón, modifica el script del Ratón. Añade un condicional "Si Puntos = 10 entonces...".
  2. Dentro de ese "Si", añade el bloque "enviar [Nivel Superado]".
  3. Ahora, ve al Fondo del escenario. Crea un nuevo script para el fondo que comience con "Al recibir [Nivel Superado]".
  4. Dentro de este script, usa el bloque "cambiar fondo a..." de Apariencia para mostrar una imagen de "¡Ganaste!".

4. Creando Ejércitos: Los Clones

¿Qué pasa si queremos crear muchos enemigos o una lluvia de objetos? Crear 50 Sprites iguales sería muy tedioso. La solución son los clones. Un clon es una copia de un Sprite que comparte sus disfraces y scripts, pero puede tener su propia posición, tamaño y comportamiento.

Bloques de Clonación

Se encuentran en la categoría Control.

  • Crear clon de [mí mismo]: El Sprite original crea una copia.
  • Al comenzar como clon: Es un bloque de evento que ejecuta el script solo en el clon, no en el original.
  • Eliminar este clon: El clon se autodestruye.
Actividad: Lluvia de Manzanas
  1. Crea un nuevo proyecto con un Sprite de una manzana.
  2. Script del Sprite Original (el "Creador"):
    • Al hacer clic en (bandera verde), esconder el original.
    • Dentro de un bucle "por siempre", esperar 1 segundo y "crear clon de mí mismo".
  3. Script de los Clones:
    • Usar el evento "Al comenzar como clon".
    • Ir a una posición X aleatoria en la parte superior de la pantalla (Y=180).
    • Mostrar el clon.
    • Usar un bucle "repetir hasta que" la posición Y sea menor que -170.
    • Dentro del bucle, "cambiar Y por -5" para que caiga.
    • Al final, "eliminar este clon" para que no se acumulen.
