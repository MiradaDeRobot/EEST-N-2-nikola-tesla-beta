Unidad 2: Creación de Componentes y Librerías

Diseño de PCB con PCB Wizard / Livewire

1. ¿Por Qué Crear Nuevos Componentes?

Aunque Livewire y PCB Wizard vienen con una extensa galería de componentes comunes, tarde o temprano te encontrarás con un componente que no está en la librería. Saber cómo crear tus propias piezas es una habilidad esencial que te da la libertad de diseñar cualquier circuito que imagines sin limitaciones.

Situaciones Comunes

  • Componentes Específicos: Necesitas usar un microcontrolador, un sensor o un circuito integrado que no viene por defecto.
  • Footprints Personalizados: Un componente existe en la librería, pero necesitas una huella física (footprint) diferente (por ejemplo, una versión SMD en lugar de THT).
  • Conectores a Medida: Quieres crear un conector con un número de pines y una disposición específicos para tu proyecto.
  • Aprendizaje y Organización: Crear tus propios componentes te ayuda a entender su estructura y a organizar tus piezas personalizadas en librerías propias.

2. Creación del Símbolo Esquemático en Livewire

El primer paso es crear el símbolo, que es la representación lógica del componente que usarás en el diagrama esquemático. No tiene por qué parecerse al componente físico, pero debe ser claro y funcional.

Usando el Editor de Símbolos

Pasos para Crear un Símbolo:

  1. Abre Livewire, ve al menú File > New > Symbol. Esto abrirá el editor de símbolos.
  2. Dibujar el Cuerpo: Usa las herramientas de dibujo de la izquierda (rectángulo, círculo, línea) para crear el cuerpo del símbolo. Un rectángulo es lo más común para circuitos integrados.
  3. Añadir Pines: Selecciona la herramienta "Pin". Haz clic donde quieras colocar un pin. Es crucial configurar cada pin correctamente:
    • Nombre del Pin: Dale un nombre descriptivo (ej. VCC, GND, TX, RX).
    • Número de Pin: Asigna el número que corresponde al pin físico del componente real. ¡Esto es vital para la conversión a PCB!
  4. Guardar el Símbolo: Ve a File > Save As. Guarda tu nuevo símbolo en una librería de usuario (puedes crear una nueva). Dale un nombre descriptivo, como "MiSensorXYZ".

3. Creación del Footprint Físico en PCB Wizard

El footprint (o huella) es el dibujo a escala real de la parte física del componente que irá en el PCB. Debe ser preciso y basarse en las medidas del datasheet (hoja de datos) del componente.

Usando el Editor de Footprints

Pasos para Crear un Footprint:

  1. Abre PCB Wizard, ve a File > New > PCB Footprint.
  2. Consultar el Datasheet: Este es el paso más importante. Busca la hoja de datos de tu componente en internet. Contendrá un diagrama con las dimensiones exactas: distancia entre pines, tamaño de los pads, etc.
  3. Colocar Pads: Selecciona la herramienta "Pad" y colócalos en el área de trabajo según las medidas del datasheet. Puedes usar la cuadrícula (grid) para ser preciso. Asegúrate de que el número de cada pad corresponda al número de pin del componente real.
  4. Dibujar la Serigrafía (Silkscreen): Usa la herramienta "Line" para dibujar el contorno del componente. Esto te servirá como guía visual al momento de soldar.
  5. Guardar el Footprint: Ve a File > Save As y guarda el footprint en una librería de usuario, idealmente con el mismo nombre que el símbolo para mantener la consistencia.

4. Vincular Símbolo y Footprint

El último paso es decirle al programa que el símbolo que dibujaste en Livewire corresponde al footprint que diseñaste en PCB Wizard. Esta asociación es lo que permite la conversión automática del esquemático al PCB.

Asociando el Footprint

  1. En Livewire, inserta tu nuevo símbolo en un esquemático.
  2. Haz clic derecho sobre el símbolo y selecciona Properties.
  3. En la ventana de propiedades, busca un botón que diga "Package" o "Footprint".
  4. Se abrirá un selector de footprints. Busca y selecciona el que acabas de crear en PCB Wizard.
  5. ¡Listo! Ahora, cuando conviertas este esquemático a PCB, PCB Wizard usará automáticamente la huella física que diseñaste.
