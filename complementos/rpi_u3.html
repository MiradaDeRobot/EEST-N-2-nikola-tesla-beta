Unidad 3: Proyectos de Visión por Computadora y Servidores

Programación con Raspberry Pi

1. Visión por Computadora con la Cámara Pi

La visión por computadora es un campo de la inteligencia artificial que entrena a las computadoras para interpretar y comprender el mundo visual. Con el Módulo de Cámara oficial, la Raspberry Pi se convierte en una plataforma excelente para experimentar con esta tecnología.

Conexión y Habilitación de la Cámara

El módulo de cámara se conecta al puerto CSI (Camera Serial Interface) de la Raspberry Pi. Una vez conectado, es necesario habilitarlo desde la configuración del sistema operativo (`sudo raspi-config` en la terminal).

Capturando una Imagen con Python

La librería `picamera` nos permite controlar la cámara de forma muy sencilla desde un script de Python.

from picamera import PiCamera
from time import sleep

# Crear un objeto cámara
camera = PiCamera()

# Iniciar la previsualización (muestra lo que ve la cámara en el monitor)
camera.start_preview()

# Esperar 5 segundos para que la cámara se ajuste a la luz
sleep(5)

# Capturar una imagen y guardarla en el escritorio
camera.capture('/home/pi/Desktop/foto.jpg')

# Detener la previsualización
camera.stop_preview()
Siguientes Pasos: Una vez que podemos capturar imágenes, el siguiente paso es procesarlas. Librerías como OpenCV nos permiten realizar tareas como detección de rostros, seguimiento de objetos o reconocimiento de colores.

2. La Raspberry Pi como Servidor Web

Gracias a que ejecuta un sistema operativo completo, la Raspberry Pi es ideal para funcionar como un servidor web de bajo consumo. Esto nos permite crear interfaces web para controlar los pines GPIO desde cualquier dispositivo en la misma red.

Creando un Servidor Web con Flask

Flask es un micro-framework de Python que hace que crear un servidor web sea increíblemente simple. Para instalarlo, abrimos una terminal y escribimos: `pip install Flask`.

Ejemplo: Controlar un LED desde el Navegador

Este script crea una página web con dos botones para encender y apagar un LED conectado al pin GPIO 17.

from flask import Flask
import RPi.GPIO as GPIO

app = Flask(__name__)

led_pin = 17
GPIO.setmode(GPIO.BCM)
GPIO.setup(led_pin, GPIO.OUT)

@app.route('/')
def index():
    return '<h1>Control de LED</h1><a href="/encender"><button>Encender</button></a><a href="/apagar"><button>Apagar</button></a>'

@app.route('/encender')
def encender():
    GPIO.output(led_pin, GPIO.HIGH)
    return 'LED Encendido'

@app.route('/apagar')
def apagar():
    GPIO.output(led_pin, GPIO.LOW)
    return 'LED Apagado'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=80)
Para ejecutarlo: Guarda el código como `servidor.py` y ejecútalo desde la terminal con `python servidor.py`. Luego, desde cualquier navegador en la misma red, accede a la dirección IP de tu Raspberry Pi.
