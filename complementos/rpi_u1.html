Unidad 1: Introducción a Raspberry Pi y Linux

Programación con Raspberry Pi

1. ¿Qué es una Raspberry Pi?

Una Raspberry Pi es una computadora de placa única (SBC - Single Board Computer) de bajo costo y del tamaño de una tarjeta de crédito. A diferencia de un microcontrolador como Arduino, una Raspberry Pi es una computadora completa: tiene un procesador más potente, memoria RAM, puertos USB, HDMI, Ethernet y ejecuta un sistema operativo completo, generalmente una versión de Linux.

Familia de Placas Raspberry Pi

No existe un solo tipo de Raspberry Pi. La fundación ha lanzado varios modelos, cada uno con diferentes características:

Modelos Principales (Ej. Pi 4, Pi 5)

Son los más potentes y con más conectividad. Ideales para usarse como una computadora de escritorio, servidor multimedia o para proyectos que requieren mucho procesamiento.

Modelos Zero

Versiones mucho más pequeñas y de menor consumo. Perfectas para proyectos portátiles o donde el espacio es crítico, como drones o cámaras compactas.

Raspberry Pi Pico

Es un microcontrolador, no una computadora. Compite directamente con Arduino y es ideal para tareas de control de hardware en tiempo real.

2. Sistema Operativo: Raspberry Pi OS (antes Raspbian)

El sistema operativo oficial es Raspberry Pi OS, una versión de Linux basada en Debian y optimizada para el hardware de la Pi. Se instala en una tarjeta MicroSD, que actúa como el "disco duro" de la computadora.

Configuración Inicial

El primer paso es "quemar" la imagen del sistema operativo en la tarjeta MicroSD usando una herramienta como "Raspberry Pi Imager". Este programa nos permite pre-configurar el nombre de usuario, la contraseña y la conexión Wi-Fi antes del primer arranque.

Una vez instalada la MicroSD, se conectan un teclado, un ratón, un monitor (vía HDMI) y la alimentación (USB-C) para iniciar el entorno de escritorio, que es muy similar al de otras computadoras.

3. La Terminal de Comandos de Linux

Aunque Raspberry Pi OS tiene una interfaz gráfica, el verdadero poder de Linux reside en su terminal de comandos. Es una interfaz de texto que nos permite interactuar directamente con el sistema operativo, siendo mucho más rápida y potente para muchas tareas.

Comandos Básicos Esenciales

Estos son algunos de los comandos que más utilizarás:

# Muestra el directorio actual
pwd

# Lista los archivos y carpetas del directorio actual
ls -l

# Cambia al directorio "Documentos"
cd Documentos

# Vuelve al directorio anterior
cd ..

# Crea una nueva carpeta llamada "Proyectos"
mkdir Proyectos

# Actualiza la lista de paquetes de software
sudo apt update

# Instala las actualizaciones disponibles
sudo apt upgrade
¿Qué es `sudo`? Es la abreviatura de "Super User Do". Se antepone a un comando para ejecutarlo con privilegios de administrador, algo necesario para instalar software o modificar archivos del sistema.
