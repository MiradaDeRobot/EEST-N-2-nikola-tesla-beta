Unidad 3: Control por Computadora y PLC

Sistemas de Control - 7º Año

1. Control de Procesos Discretos

Un proceso discreto es aquel en el que las variables de estado cambian en instantes de tiempo específicos, en lugar de hacerlo de forma continua. El control se basa en eventos y secuencias lógicas (ON/OFF, abierto/cerrado, etc.).

La Computadora como Elemento de Control

La computadora (ya sea una PC industrial o un PLC) reemplaza la lógica de relés cableada por un programa de software. Esto ofrece una flexibilidad inmensa, permitiendo modificar la lógica de control sin cambiar el cableado físico.

Arquitectura General: El sistema consta de una CPU, memoria, interfaces de entrada/salida (E/S) para conectar sensores y actuadores, y una interfaz hombre-máquina (HMI) para la supervisión.

2. Control mediante PLC

El Controlador Lógico Programable (PLC) es el dispositivo por excelencia para el control de procesos discretos en la industria. Es una computadora robusta, diseñada para soportar ambientes hostiles y para ser programada de forma intuitiva por personal técnico.

Arquitectura y Tipos de PLC

  • PLC Compacto: La CPU, la fuente de alimentación y los módulos de E/S están integrados en una sola carcasa. Son ideales para aplicaciones pequeñas y medianas.
  • PLC Modular: La CPU, la fuente y los módulos de E/S son componentes separados que se montan sobre un riel (rack). Permite una gran flexibilidad y escalabilidad para sistemas complejos.

Programación de PLC

Existen varios lenguajes de programación estandarizados (IEC 61131-3):

Diagrama de Escalera (Ladder)

El más popular. Su simbología se basa en los circuitos de control con relés.

Diagrama de Bloques de Funciones (FBD)

Utiliza bloques que representan funciones lógicas y aritméticas, conectados entre sí.
