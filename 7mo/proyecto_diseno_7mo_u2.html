Unidad 2: Planificación y Diseño

Proyecto y Diseño Electrónico - 7º Año

1. Planificación y Seguimiento del Proyecto

La planificación es el proceso de definir las acciones y recursos necesarios para alcanzar los objetivos del proyecto. Un buen plan es la base para el éxito. El seguimiento es el control continuo del avance del proyecto para compararlo con lo planificado y tomar acciones correctivas si es necesario.

Análisis y Relevamiento

Esta es la fase de investigación inicial. Antes de planificar, se debe realizar un relevamiento exhaustivo para recopilar toda la información necesaria:

  • Requerimientos del Usuario: ¿Qué debe hacer exactamente el producto final?
  • Investigación de Tecnologías: ¿Qué componentes, sensores o microcontroladores son los más adecuados?
  • Análisis de Antecedentes: ¿Existen proyectos similares? ¿Qué se puede aprender de ellos?

2. Definición de Etapas y Tiempos

Un proyecto complejo se debe dividir en partes más pequeñas y manejables. Estas partes se conocen como etapas o fases.

Diagrama de Gantt

El Diagrama de Gantt es la herramienta de planificación por excelencia. Es un gráfico de barras horizontales que muestra las diferentes tareas del proyecto a lo largo del tiempo. Permite visualizar:

  • La fecha de inicio y fin de cada tarea.
  • La duración de cada tarea.
  • Las dependencias entre tareas (qué tarea debe terminar antes de que otra pueda empezar).
  • El progreso actual en comparación con el plan original.

3. Selección de Herramientas

La elección de las herramientas adecuadas (tanto de software como de hardware) es una decisión estratégica que impacta directamente en la calidad, el costo y el tiempo de desarrollo del proyecto.

Herramientas de Software y Hardware

Software

  • Diseño CAD/EDA: KiCad, Eagle, Altium Designer (para esquemáticos y PCB).
  • Simulación: LTspice, Proteus, Multisim (para verificar circuitos).
  • Programación (IDE): Arduino IDE, MPLAB X, VS Code (para escribir el firmware).
  • Diseño Mecánico: Fusion 360, SolidWorks (para diseñar gabinetes y piezas 3D).

Hardware

  • Instrumental de Medición: Multímetro, osciloscopio, fuente de alimentación, generador de funciones.
  • Herramientas de Montaje: Soldador, estación de soldado, herramientas manuales.
  • Prototipado Rápido: Impresora 3D, cortadora láser.

4. Diseño y Elaboración de los Elementos

Esta es la fase de ejecución donde las ideas y los planes se convierten en un producto tangible. Es un proceso iterativo que combina el diseño electrónico, el diseño de software y el diseño mecánico.

Flujo de Diseño Integrado

  1. Diseño del Circuito Esquemático: Se dibuja el diagrama lógico en el software CAD.
  2. Simulación del Circuito: Se verifica el comportamiento de las partes críticas del circuito.
  3. Desarrollo del Firmware: Se escribe y se prueba el código que correrá en el microcontrolador.
  4. Diseño del PCB: Se realiza el layout de la placa, se ubican los componentes y se rutean las pistas.
  5. Diseño Mecánico: Se diseña el gabinete o las partes mecánicas, asegurando que el PCB y los demás componentes encajen correctamente.
  6. Prototipado y Ensamble: Se fabrica el PCB, se imprimen las piezas 3D y se sueldan los componentes.
  7. Pruebas y Depuración: Se verifica que el prototipo completo funcione según lo esperado y se corrigen los errores.
